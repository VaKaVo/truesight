# TrueSight
TrueSight is a Dota 2 surface overlay that shows the real stats of each player in a match — like integrity, win rate, medals, and top heroes — right at their peak.

1. What does it show?
   - 🎖 Guardian Medal → Immortal with color
   - ❤️ Decency Behavior score with color indication
   - 📊 All-time win rate%
   - ⚔️ W / L Total wins and losses

Colors: 🟢 good · 🟡 average · 🔴 bad · ⬜ profile hidden

## Installation
**Ready EXE (Windows)**

1. Download the archive from Releases
2. Extract to any folder
3. Run TrueSight.exe

## From the source code:
```
git clone https://github.com/VaKaVo/truesight
cd truesight
pip install requests
python tracker.py
```
